# WholesaleAll
Automatically trades all configured coins to destination coins

  For example:

    "sell_all": {
      "eth": {
        "usdt": 100
      },
      "rvn": {
        "etc": 50
      }
    }

### Config for deposits:
* Trade 100% of deposited ETH to USDT
* Trace 50% of RVN via USDT to ETC
